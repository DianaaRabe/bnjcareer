# 🚀 BNJ Career - Plateforme d'Accompagnement IA

BNJ Career est une plateforme moderne et intuitive conçue pour révolutionner l'accompagnement vers l'emploi. Alliant puissance de l'IA et suivi humain, elle offre aux candidats un écosystème complet pour booster leur carrière.

## ✨ Fonctionnalités Clés

### 📄 CV & Matching Intelligent
*   **Génération de CV IA** : Extraction et structuration automatique de vos données depuis un PDF vers un format professionnel.
*   **Matching Offre/Profil** : Analyse instantanée de la compatibilité entre votre CV et une offre d'emploi avec un score et des recommandations détaillées.

### 🔍 Automatisation de la Recherche
*   **Scrapper Multi-Plateformes** : Recherche centralisée d'offres sur Indeed (et prochainement Monster, Welcome to the Jungle, Apec).
*   **Candidature en un clic** : Génération de messages de motivation personnalisés et liens directs vers les plateformes de recrutement.

### 📅 Accompagnement & Gamification
*   **Sessions de Coaching** : Réservation d'ateliers collectifs ou de sessions 1v1 avec nos experts.
*   **Système d'Objectifs** : Progression gamifiée pour rester motivé tout au long de votre recherche.

### 💬 Communication & Ressources
*   **Messagerie Intégrée** : Chat direct avec votre coach attitré et accès à la communauté BNJ.
*   **Bibliothèque de Savoir** : Accès privilégié à des guides, vidéos et replays d'ateliers.
*   **Assistant IA BNJ** : Un chatbot expert disponible 24/7 pour répondre à toutes vos questions carrières.

## 🛠 Stack Technique

*   **Framework** : [Next.js 14](https://nextjs.org/) (App Router)
*   **Langage** : TypeScript
*   **Styling** : Tailwind CSS
*   **Icônes** : Lucide React
*   **Backend / APIs** :
    *   CV Data Extraction API
    *   Job Matching Score API
    *   Apify (Indeed Scrapper)

## 🚀 Installation & Lancement

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/votre-repo/bnj-career.git
    cd bnj-career
    ```

2.  **Installer les dépendances** :
    ```bash
    npm install
    ```

3.  **Configurer l'environnement** :
    Créez un fichier `.env` dans `apps/web` et ajoutez vos clés API (APIFY_API_TOKEN, etc.).

4.  **Lancer en mode développement** :
    ```bash
    npm run dev
    ```
    Accédez ensuite à [http://localhost:3000](http://localhost:3000).
